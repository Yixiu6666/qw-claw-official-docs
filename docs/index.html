常见问题

关于企微管家Claw的常见疑问解答

企微管家Claw 和 OpenClaw 是什么关系？
OpenClaw 是开源的 AI 智能体框架。企微管家Claw 是基于 OpenClaw 做的成品化封装——已处理好企微适配、预装了企微 Skills、做了可视化安装界面，拿来就能用。
数据安全怎么保证？
100% 本地部署，所有数据、记忆、文件留存在用户自己的电脑上。提供四级权限模式（Full / Coding / Chat / Minimal），可以把AI限制为只聊天、不读写文件。
需要技术人员才能安装吗？
不需要。全程不写一行代码，全是页面操作——安装包双击安装、模型配置填表格、企微扫码即连。运营、销售、人事都能自己装好。
支持什么系统？
支持 macOS（Apple/Intel）和 Windows。已安装了开源版 OpenClaw 不会冲突，直接安装即可。
模型需要额外付费吗？能换吗？
安装包本身免费。推荐使用微盛大模型服务：微信扫码登录后注册即送百万 Token，支持智谱 GLM-4.7-Flash 免费使用。也可自行填写 API Key 接入其他模型平台。
个人微信能接入吗？
已支持！微信官方推出了「ClawBot」插件，在设置页扫码即可连接。企微 + 微信双通道可同时在线。
手机上能用吗？
企微管家Claw 安装在电脑上。但接入后，在手机端的企微和微信里都能直接和 AI 对话——电脑是"大脑"，手机是随身入口。
部署在一台电脑上，能全公司用吗？
可以。连接到企微智能机器人后，可以在企微通讯录里开放给全部员工使用，也可设置可见范围限定部门。
企微管家Claw 和微盛 SCRM 什么关系？
二者是互补关系。SCRM 负责客户运营（标签、群发等），Claw 负责 AI 智能体能力。两者已通过 SCRM Skills 深度打通——AI 生成内容 → 一键调用 SCRM 创建任务。
支持哪些大模型？
支持企微管家Claw账户余额、OpenRouter、腾讯云 Coding Plan、DeepSeek等。优先推荐微盛模型服务，扫码登录后即可直接调用。更多模型平台在陆续接入中。
