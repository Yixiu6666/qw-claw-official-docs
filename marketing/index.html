常见问题

客户/渠道商最常问的问题

这些问题在直播和客户沟通中被反复提及，销售同学务必熟悉答案

企微管家Claw 和 OpenClaw 是什么关系？
OpenClaw 是开源的 AI 智能体框架，相当于"刚从水里抓出来的龙虾"。企微管家Claw 是基于 OpenClaw 做的成品化封装——已经处理好企微适配、预装了企微 Skills、做了可视化安装界面，相当于"一道做好的菜"，拿来就能用。
数据安全怎么保证？会不会有泄露风险？
100% 本地部署，所有数据、记忆、文件留存在用户自己的电脑上。安全的关键不是"接不接企微"，而是权限怎么配。我们提供四级权限模式（Full / Coding / Chat / Minimal），可以把AI限制为只聊天、不读写文件。比网上裸跑开源版安全得多。
能在外部群/对客户使用吗？
当前主要支持企业微信内部场景——内部单聊、内部群聊、内部群 @机器人。外部群和直接对客自动服务暂不支持。这是因为企业微信官方对外部群的接口权限限制，不是我们产品的限制。
需要技术人员才能安装吗？
不需要。全程不写一行代码，全是页面操作——安装包双击安装、模型配置填表格、企微机器人复制粘贴。不懂技术的运营、销售、人事都能自己装好。如果企业希望省心，也可以购买"企业启用服务包"由我们协助。
支持什么系统？已经装了开源版怎么办？
支持 macOS（Apple/Intel）和 Windows。已经安装了开源版 OpenClaw 不会冲突，直接安装企微管家Claw 即可，两者独立运行。
模型需要额外付费吗？能换吗？
安装包本身免费。优先推荐直接使用微盛大模型服务：在“设置” -> “我的帐户”里微信扫码授权登录，注册后即可领取百万 token，并支持智谱 GLM-4.7-Flash 免费使用。登录完成后，需要在“大模型设置”里手动选择“微盛 Ws-Claw（帐户余额）”和具体模型，再点击“保存大模型设置”才会生效。若使用该模式，则由微盛统一提供中转服务和计费，背后接入各厂商官方大模型 API；当然也仍支持自行填写 API Key 接入其他模型平台。
需要先建数据库吗？
不需要。安装好就能直接使用，不需要任何数据库环境搭建。
企微管家Claw 和微盛 SCRM 是什么关系？
二者是互补关系，不是替代关系。SCRM 负责客户运营体系（标签、群发、朋友圈任务等），企微管家Claw 负责 AI 智能体能力。现在两者已通过 SCRM Skills 深度打通——AI 生成内容 → 一键调用 SCRM 创建任务 → 统计督办。
手机上能安装吗？
手机端不能直接安装。企微管家Claw 需要安装在 Mac 或 Windows 电脑上。但安装并接入后，在手机端的企业微信和个人微信里都能直接和 AI 对话——电脑是"大脑"，手机企微和微信都是随身入口。
个人微信能接入吗？
已支持！微信官方推出了「ClawBot」插件，在企微管家Claw 设置页点击「安装/绑定微信」，手机微信扫码即可连接。连接后，在微信里给 ClawBot 发消息就能和龙虾对话，所有功能和企微通道一致。企微 + 微信双通道可同时在线。
企微官方的 OpenClaw 插件和微盛安装包有什么区别？
企微官方推出的是 OpenClaw 的一个channel 插件，帮已装好 OpenClaw 的用户连接企微机器人。而微盛安装包是一键安装并部署 OpenClaw + 自动集成企微 channel 插件 + 预装企微 Skills 的完整方案。简单说：官方插件只解决"连接"，我们解决"安装+连接+能力扩展"全流程。
部署在一台电脑上，能全公司使用吗？
可以。在一台电脑上本地部署后，连接到企微智能机器人，这个机器人可以在企微通讯录里开放给全部员工使用，也可以设置可见范围，只对特定部门或个人可见。
能做AI客服/主动销售吗？
如果是企微内部的智能问答助理，可以做到。但如果是模拟真人跟外部客户主动销售对话，企业微信官方能力暂不支持。企微对客的智能客服场景，建议使用微盛AI企微管家的「智能回复」功能，可结合企业知识库生成回复。
支持华为鸿蒙/Linux服务器吗？
华为鸿蒙系统近期适配中，敬请期待。Linux 服务器部署暂不在安装包支持范围内，但技术团队可以单独沟通定制方案。当前优先支持 Windows 和 macOS。
内网环境能用吗？
可以。OpenClaw 本身支持内网部署。但大模型调用需要网络，如果是纯内网环境，需要自行解决大模型的网络连通问题，或者部署本地大模型。
支持哪些大模型？阿里百炼能用吗？
当前支持 微盛 Ws-Claw（帐户余额）、OpenRouter、腾讯云 Coding Plan（含智谱/Kimi/Minimax）、DeepSeek 等。优先推荐使用微盛模型服务，微信扫码登录后即可直接调用智谱等官方模型，其中注册即送百万 token，并支持智谱 GLM-4.7-Flash 免费使用。阿里百炼目前暂不支持，更多模型平台在陆续接入中。
安装包免费吗？你们靠什么盈利？
安装包完全免费，不靠安装包盈利。微盛的核心产品是企业微信 SCRM 工具。企微管家Claw 的定位是帮助更多企业用好企业微信+AI，企业如需专人协助部署和场景落地，可选购启用服务包。
界面是中文的吗？
支持中文。系统内可设置语言为中文，操作界面和配置流程都对中文用户友好。
