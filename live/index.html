开场

云御：

各位老板下午好，欢迎来到我们的直播间。我是今天的主持人云御，旁边是一休，是我们产品负责人。

最近大家应该都知道，全网都在刷"养龙虾"这个事情。企业微信上周也支持了类似玩法，我们第一时间做了封装和适配，所以有了今天要给大家演示的企微管家Claw。

今天这场直播我们不讲虚的，就讲一件事情：怎么在你的企业微信上，把企微管家Claw 真正用起来，让你的企业微信从以前很多事情都靠人工、很累、很碎的状态，升级成一个能让 AI 帮你干活的入口。

我们今天带来的不是概念，是解决方案。会有现场实操，会有真实案例，也会讲避坑。大家听完以后，回去就能落地。

为什么做这个产品

云御：

大家这段时间应该看过很多 OpenClaw 相关教程，但说实话，很多教程对普通人不友好。要写代码、要做适配、要配环境，对很多运营、销售、人事或者老板来说，门槛还是挺高的。

我们做企微管家Claw，核心就想解决这个问题。我们想把它做得足够简单，简单到什么程度？就是不懂代码、不懂 API 的人，也能通过点几下鼠标，把企业微信和 OpenClaw 连接起来。

这不是一句宣传话术，是真的想把门槛降下来。

大家现在最常见的问题

云御：

我们最近接触客户，大家最常见的问题其实就这些：

装不上，版本不对；

Windows 不知道怎么配；

本地文件喂不进去；

改了配置，但机器人效果没变化；

没有长期记忆；

电脑性能差一点就跑崩；

不知道该选哪个模型，也不知道怎么买、怎么配。

所以今天这场直播，不是泛泛聊 AI，而是把这些实际问题拆开给大家看。

为什么一定要接企业微信

云御：

很多人都在讲怎么本地部署 OpenClaw，但我们更关心的是，为什么要把它接进企业微信。

因为企业微信是大家每天都在打开的地方。你如果还要再开一个网页、再进一个平台、再切一个窗口，那绝大多数人其实用不起来，或者坚持不下来。

但如果把 AI 直接放进企业微信，让它变成一个你随手就能找到、随手就能叫它做事的助理，那事情就不一样了。它就不只是一个聊天工具，而是开始进入业务流程、执行流程、回传结果。

微盛内部怎么用

云御：

我们公司内部其实已经从上周开始在全员用这个安装包做部署，并且接到了企业微信里。以前大家碰到问题，可能要开豆包、开元宝，等结果出来，再复制回企业微信。中间一旦任务复杂、时间长，人就切走了，甚至会忘掉原来要干什么。

现在接进企微以后，大家就是在企微里直接找"小龙虾助理"，提问、下任务、拿结果，一条链就走完了。

而且我们也发现，原来不怎么用 AI 的同事，现在也开始慢慢把自己的龙虾养起来了。它会越来越了解业务，也会越来越了解个人习惯。

OpenClaw 和普通聊天机器人有什么区别

云御：

像豆包、元宝这种，更像是问一句答一句。

但 OpenClaw 不是，它是"长了手"的。它不只是回答你，它能调用工具、能读文件、能查历史、能执行动作，能真正帮你干活。所以我们觉得，把这种能力接进企业微信，价值很大。

安装演示开始

一休：

好，下面我从零开始给大家演示一下怎么安装我们的企微管家Claw。

这个安装包支持 Windows 和 Mac，我今天演示的是 Mac 版。大家拿到安装包以后，双击打开，像安装普通软件一样，把它拖到应用程序文件夹里就可以了。

如果第一次打开的时候系统提示"无法打开"或者"已阻止打开"，这是正常情况。很多外部下载的软件都会这样。你去系统设置里找到"隐私与安全性"，允许它继续打开，再输入开机密码，就可以进到部署界面。

第一次部署怎么配

一休：

进到部署界面以后，要先配大模型的信息。我们现在支持的模型服务商挺多的，比如腾讯云 CODING plan、DeepSeek、Kimi、Minimax、智谱，还有 OpenRouter 这些。

以腾讯云 CODING plan 为例，它比较适合大部分普通办公场景，便宜、够用、稳定。你只要把模型服务商、API Key、请求地址、模型名这些按示例填进去，然后点"保存并自动部署"，它就会自动开始跑。

这一步其实已经被我们尽量做简单了，不需要你懂底层代码。

官方版和我们这个版本有什么区别

一休：

你可以把开源版理解成一只刚从水里抓出来的龙虾，后面怎么做、怎么炒、怎么调味，都要你自己来。

但企微管家Claw 更像是一道已经给你做好的菜。很多配套的步骤、企业微信的适配、常用的企微 skills，我们已经提前给你处理好了。你拿到手以后，能更快地上手，能更快地用起来。

安全问题怎么理解

一休：

直播里大家问得最多的就是安全。

我的看法是，真正的风险不在于"接不接企业微信"，而在于你的权限怎么配。开源裸跑，很多时候才更危险。我们这个安装包做了比较可视化的权限管理，而且我们自己内部已经用了挺长时间了。

它可以是企业统一用的机器人，也可以是个人专属的机器人。如果你担心本地隐私数据，那就把权限调低，比如只保留聊天能力，不让它读写文件、不让它执行命令。后面我们也会做更细的权限控制，比如只放开某个文件夹。

模型是不是能随时换

一休：

可以。你一开始配的是哪个模型，后面如果用着觉得不合适，想换成别的模型，直接在后台改配置就行，不需要重新安装整个程序。

接入企业微信

一休：

部署完成以后，进入 channel 页面，就能直接看到企业微信插件。这个是因为我们已经把企业微信官方 channel 集成到安装包里了。

接下来要做的事情也很简单：

去企业微信通讯录里找到"智能机器人"；

创建一个新的机器人；

选手动创建；

再选 API 模式；

把机器人名称、头像这些基础信息填好。

现在企业微信出了长连接方式，这一点特别重要。以前很多企业卡在要域名、要 IP，现在不需要了。你只要拿到 bot ID 和 secret，把这两个值复制到我们后台，更新一下配置，就能启用。

然后回到企业微信，给机器人发一句"你好"，它返回配对信息，就说明已经接通了。

整个过程不写一行代码，全部都是页面操作。

接通以后能干什么

云御：

接通以后，其实就是从"装好了一个工具"变成了"企业微信里多了一个智能助理"。

后面重点就不是"能不能装"，而是"它能帮你做什么"。

场景一：定时任务

一休：

第一个很直观的场景，就是定时任务。

比如说你让它每天上午 10 点，联网搜一下 OpenClaw 的热点资讯，整理完以后推给你。那你第二天上班的时候，企微里就已经躺着一份整理好的内容了。

再比如中午提醒吃饭、每小时提醒喝水、下班提醒打卡、重要节日提醒准备礼物或者发祝福，这些它都能做。

而且不是死板地重复一句提醒，它会结合上下文，生成不同的表达，还会记住你之前的反馈。

场景二：品牌内容生成

一休：

第二个场景，是帮你做内容。

比如你让它搜一个品牌的信息，然后生成这个品牌一整年的节假日朋友圈营销文案，再同步到企业微信智能表格里。以前这种事情靠人工做很慢，现在你只要给它一句任务，它就能先搜、再整理、再写、再同步。

这里用到的是我们预置的企微 skills。安装好相关技能以后，龙虾就不只是"会说"，而是真的能把结果写进企微表格里。

这样后面运营、销售、客户成功都可以基于这张表去继续执行。

场景三：在线文档同步

一休：

第三个场景，是把调研结果同步到企微在线文档。

比如你让它搜某个品牌、某个竞品、最近 7 天的更新信息，然后整理成报告。它做完以后，不是给你扔一段聊天回复，而是直接返回一个企微在线文档链接。你点开就能看，也能转发给同事。

直播里我们现场演示了搜"微盛企微管家Claw"的公开信息，然后同步成企微文档。出来的效果其实挺不错的，定位、能力、案例、链接都整理进去了。

场景四：做 PPT

一休：

还有一个很实用的场景，就是做 PPT。

你装好对应技能以后，直接在企业微信里告诉它：帮我做一个某某主题的 PPT，或者基于我本地这份材料做一套产品介绍 PPT。它就可以结合联网信息和本地资料，直接给你生成在线 PPT。

后面还可以进一步把演讲稿、讲解话术一起生成出来。

场景五：调用其他智能体

一休：

还有更进阶一点的玩法，就是让龙虾去调用别的平台上的智能体，比如扣子。

很多团队以前已经在扣子上做过智能体了，但入口很分散。现在如果把这些能力封成 skill，接进企微管家Claw，那你以后就只需要在企业微信里说一句话，龙虾就能替你调用对应智能体。

我们直播里也演示了让它调用扣子的营销话术智能体，生成企微管家Claw 的产品推广话术。这个思路是很重要的：未来不是你去找一堆工具，而是企业微信里这个助理替你调度这些工具。

和 SCRM 的关系

云御 / 一休：

企微管家Claw 不是 SCRM，但它可以和 SCRM 联动。

举个例子：

先让龙虾生成朋友圈营销文案；

再调用 SCRM 的能力创建一个朋友圈任务；

确认没问题以后下发给销售；

最后再让龙虾去统计谁完成了、谁没完成，并继续提醒。

这就不是单纯"写文案"了，而是进入了业务执行闭环。

直播里反复回答的几个问题

云御：

Windows 可以装。

已经装了开源版也不冲突。

外部群目前还不支持，主要支持企业微信内部场景。

这场直播最核心的一句话

云御：

我们今天真正想表达的是：企业微信不只是聊天软件，它会变成一个 AI 的工作入口。

龙虾也不只是一个陪你聊天的机器人，而是一个能记住你、能调用工具、能持续做事的数字助理。

而企微管家Claw 想做的，就是把这件事从技术玩家的能力，变成普通岗位也能真正上手的能力。

一、CTO出场 — 五层安全架构深度讲解（全新）

3.11仅简单提及权限分级和安全概念，3.12由CTO系统性地阐述了企业级五层安全体系：

  1. 用户访问层: 用户身份识别与准入控制
  2. 访问控制层: 零信任架构、多因素认证（MFA）、身份鉴权
  3. 内容审计层: 输入/处理/输出三层过滤机制，Prompt注入检测（全新概念）
  4. 网关层: Skill沙箱隔离、技能市场审核机制——公开skill约10%存在安全隐患（20万+公开skill中），禁止未授权skill安装
  5. 数据与网络层: 审计日志180天留存、SSRF防护、域名白名单

二、工信部「六要六不要」合规详细对照（全新）

CTO逐条将工信部3月11日发布的AI使用「六要六不要」安全指南与企微管家Claw的安全措施进行了一一对照映射，展示了每条要求的具体落地方案。3.11未涉及此内容。

龙虾接进企微了，但它现在还是一个"白板"。就像你买了一台新手机但没装App，只能打电话发短信。龙虾也一样，需要安装Skills——每个Skill就是一个能力模块，装什么就会什么。

推荐安装的Skills：

Skill名称功能说明适用场景
企微在线文档同步生成报告并同步到企微在线文档写报告、做方案

安装方法很简单：点击"企微Skills"，找到想装的Skill，点"安装"。几秒钟就好，安装完成后龙虾自动获得该能力，不需要额外配置。

4.2 设置AI身份和性格

Skills装好之后，还有一个重要步骤——给龙虾一个身份和性格。

很多AI工具每次聊天都不记得你之前说过什么，这就是没有"记忆"。龙虾不一样：你告诉它你是谁、喜欢什么风格、怎么称呼，它会一直记住。

可以设置：

  • 叫你的名字或昵称（比如"御哥"而不是"用户"）
  • 用幽默风趣的风格说话
  • 先给结论再讲细节
  • 遇到不确定的事情先问你
  • 用特定的Emoji

设置方式：直接在聊天界面输入以下信息：

请记住以下设置：

1. 名字：xx

2. 形象：靠谱、高效、话少事多

3. 风格：幽默风趣、逻辑清晰、不绕弯

4. Emoji：🦞

5. 称呼我：御哥

6. 核心性格：先给出解决方案，再补充细节；提问不够深入时主动追问

金句："你培养它，它就会越来越懂你。就像一个真正有记忆的助理。"

4.3 场景一：定时任务

第一个最容易上手、最直观感受到价值的场景。

你有没有这种经历——想每天固定时间做某件事，但经常忘？想每天看行业新闻但总是忘了刷App；想每小时站起来活动一下但一忙就忘了；想按时喝水但每次都是渴了才想起来。

演示：在企微里给龙虾发

"xx，请帮我每天上午10点联网搜索AI行业最新资讯，整理后推送给我"

从明天开始，你每天早上上班，打开企业微信，就已经有一份整理好的行业资讯等你了。和闹钟不一样——闹钟只会"滴"一声提醒你去看新闻，龙虾是帮你搜好、整理好、推送给你。假设每天花15分钟筛选信息，一年就是约90个小时，这些时间可以让龙虾帮你省下来。

更多定时任务示例：

任务类型示例话术效果
喝水提醒"每小时提醒我喝水，并记录饮水量"定时提醒+自动记录
吃饭提醒"每天12点提醒我吃饭"防止忙忘了吃饭
打卡提醒"每天18:30提醒我下班打卡"防止忘打卡
竞品监控"每天15点搜索竞品XX最新动态，汇总发给我"自动监控竞品
节日提醒"提前3天提醒我即将到来的节日"提前准备节日活动

注意事项：定时任务需要电脑保持开机状态，因为龙虾是部署在你本地电脑上的。建议设一些工作时间段的任务。后续会优化支持云端调度。

金句："它不是死板地重复一句话，会结合上下文生成不同的表达。"

4.4 场景二：品牌内容生成 + 智能表格同步

假设老板安排你："把全年节假日的营销文案都写一下"。春节、元宵、清明、五一、端午、中秋、国庆、双十一、双十二、圣诞节……传统做法是打开日历一个一个节日看，每个节日想创意、写文案，写完还要整理成表格，可能要一周时间。

演示：在企微里给龙虾发

"搜索微盛公司信息，生成2026年全年节假日朋友圈营销文案，同步到企微智能表格"

龙虾会：联网搜索品牌信息 → 基于信息生成文案 → 同步到企微智能表格。几分钟出第一版，每个节日都有，格式规范，可以直接复制发布。以前运营写全年文案可能要一周，现在几分钟出第一版，还直接写进企微表格了。

金句："以前运营写全年文案可能要一周，现在几分钟出第一版，还直接写进企微表格了。"

4.5 场景三：在线文档同步

假设周五下午领导突然找你："下周一给我一份竞品分析报告，详细一点的。"传统做法：搜竞品动态、整理成报告、排版美化、发给领导——整个流程两天。

演示：在企微里给龙虾发

"搜索企微管家最近7天的产品更新和行业动态，整理成竞品分析报告，同步到企微在线文档"

龙虾工作流程：联网搜索信息 → 整理成报告结构 → 生成企微在线文档。文档包含：概述、产品动态、行业趋势、竞品对比、机会点分析、建议。内容全面，格式规范，可以直接发给领导预览，还能转发给同事协作。

金句："领导让你两天出一份竞品报告？现在在企微里跟龙虾说一句话，几分钟后就有文档。"

4.6 场景四：生成PPT

你们最讨厌的工作任务是什么？做PPT。为什么讨厌？要找模板、要排版、要写内容、要调格式。一份像样的PPT可能要半天甚至一天。

演示：在企微里给龙虾发

"帮我做一个'微盛企微管家产品介绍'的PPT，用于客户高层汇报，风格专业简洁"

龙虾收到指令后会：理解需求 → 联网搜索产品信息 → 生成PPT大纲 → 生成每一页内容 → 输出在线PPT链接。生成的PPT包含：封面、目录、产品概述、核心功能、客户案例、合作模式、联系方式。

进阶玩法：还可以让龙虾生成每一页的讲解话术。发一句"帮我生成这份PPT每一页的讲解话术"，龙虾会根据每一页内容生成演讲稿。

4.7 场景五：调用其他智能体

很多团队已经在扣子、Dify这些平台上做过智能体了，但入口很分散——用的时候要打开扣子网页、Dify网页、某个小程序，每个智能体一个入口，很麻烦。

现在可以把这些能力接到企微管家Claw里，在企业微信里说一句话，龙虾就能帮你调用对应的智能体。

演示：

"调用扣子营销话术智能体，帮我生成'企微管家AI功能'的推广文案，用于朋友圈发布"

龙虾会：识别意图 → 传递参数给扣子 → 获取扣子智能体的输出 → 返回给你。你不需要打开扣子网页，不需要登录，在企微里就搞定了。

金句："未来不是你去找一堆工具，而是企业微信里这个助理替你调度这些工具。"

4.8 场景六：与SCRM联动（未来规划）

企微管家Claw不是SCRM，不会取代SCRM。它是SCRM的"搭档"，是连接你和各种系统的"桥梁"。

联动后的流程：

  1. 你在企微里跟龙虾说："帮我生成这周的朋友圈营销内容，并创建SCRM任务下发给销售"
  2. 龙虾自动：生成文案 → 创建任务 → 下发给销售
  3. 每天自动统计完成情况，推给你
  4. 对未完成的销售自动提醒

以前你要手动做每一步，现在只需要说一句话。这就不是单纯"写文案"了，而是进入了业务执行闭环。

企微管家Claw的定位很清晰：它不是要替代任何系统，而是让各个系统之间的协作更顺畅。就像智能家居：空调还是空调，电视还是电视，但你可以用手机App统一控制。龙虾就是那个"控制中心"。

5.5 安全类问题

问题回答
数据安全吗？100%本地部署，数据全部留在你的电脑上
权限怎么控制？提供四级权限：Full / Coding / Chat / Minimal
会泄露隐私吗？安全的关键是权限配置，可设置为仅聊天模式
裸跑开源版安全吗？比封装版更危险，建议使用经过安全检测的安装包

四级权限说明：

  • Full（完全控制）：能做任何事情，包括读写文件、执行代码、改设置。适合完全信任的用户。
  • Coding（编码模式）：能执行代码，但不能改系统设置。适合开发场景。
  • Chat（聊天模式）：只能聊天，不能读写文件。适合只想用龙虾聊天的用户。
  • Minimal（最小权限）：只能回复消息。最安全的模式。
